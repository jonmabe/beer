# Half Blind Raven Table Saison v2 — Brew Day Guide

**Date:** March 7, 2026
**System:** Ss Brewtech SVBS
**Method:** Full-volume no-sparge BIAB
**Yeast:** Mangrove Jack's M29 French Saison (dry)

---

## 🔧 Setup

1. Install **basket seal** onto ¾" TC flange in kettle base
2. Moisten **pump filter gasket**, install **pump filter** in base (cover drain hole)
3. Attach **recirc tube** to ¾" TC with clamp + gasket
4. Slide **RIMS diffuser** onto recirc tube (~½" on)
5. Connect **knockout tubing** (3/8" silicone) to knockout TC — open end in Star San bucket

## 🧹 Pre-Sanitize Lines

1. Close all 3 valves (Recirc, Whirlpool, Knockout)
2. Add 2 gal Star San solution to kettle
3. Open Recirc + Whirlpool → Pump ON → recirculate 2-3 min
4. Close Recirc + Whirlpool → Open Knockout → drain
5. Pump OFF → close all valves
6. Place knockout tube end back in Star San

## 💧 Fill & Heat Strike Water

1. Close all valves
2. Pour in **6.23 gal distilled water** (full volume, no sparge)
3. Add **1 tsp CaCl₂** (5.5g) → stir to dissolve
4. Open Recirc valve to purge air, then close Whirlpool (Recirc stays open)
5. Insert **mash basket** — push down to seat
6. Pump ON → slowly open Recirc → confirm flow through recirc tube
7. Controller: **Mash mode → 163°F** → Heater ON
8. Lid on mash basket while heating

| Valve | Position |
|-------|----------|
| Recirc | OPEN |
| Whirlpool | CLOSED |
| Knockout | CLOSED |
| Pump | ON |
| Heater | ON (163°F) |

## 🌾 Mash In

1. Water hits 163°F → **Pump OFF, Heater OFF**
2. Lower set temp to **156°F**
3. Remove lid → slowly pour in grain → stir gently to break clumps
4. Lower RIMS diffuser to ~2" above grain bed
5. Pump ON → slowly open Recirc → watch flow
6. Monitor fluid level — dial back recirc if it rises too fast
7. Heater ON → controller holds 156°F
8. Lid on → **mash 60 minutes**

⚠️ **Stuck mash?** If water pools above grain: kill heater + pump → stir grain bed → restart

| Valve | Position |
|-------|----------|
| Recirc | OPEN (adjusted) |
| Whirlpool | CLOSED |
| Knockout | CLOSED |
| Pump | ON |
| Heater | ON (156°F) |

## ⬆️ Mash Out & Drain

1. After 60 min → raise temp to **168°F** → hold **10 min**
2. Pump OFF, Heater OFF
3. Slowly lift mash basket → rest on support ledges inside kettle
4. Drain 10-15 min
5. Move basket over 5-gal bucket to catch drips → add back to kettle
6. Remove basket, set aside

## 🔥 Boil (60 minutes)

1. Close Recirc valve, Pump OFF
2. Switch to **Boil mode → 100% power**
3. Watch for boil-over → reduce to **~60%** at rolling boil
4. Start 60-min timer at first rolling boil

### Addition Schedule

| Time Left | Addition |
|-----------|----------|
| **60 min** (start) | 0.5 oz **Styrian Goldings** |
| **15 min** | 1 tsp **Irish Moss** |
| **10 min** | 0.5 oz **Hallertau Blanc** + **0.4 lb sugar** (stir to dissolve) |
| **5 min** | **Crushed coriander** (0.5 tsp) + **orange zest** (1 orange, no pith) |
| **0 min** | Kill heat |

| Valve | Position |
|-------|----------|
| All valves | CLOSED |
| Pump | OFF |
| Heater | ON (Boil ~60%) |

## 🌀 Whirlpool

1. Heater OFF
2. Drop **immersion chiller** into hot wort (sanitizes in place)
3. Open Whirlpool valve → Pump ON → **whirlpool 10 min**
4. Pump OFF → close Whirlpool valve
5. **Rest 10 min** (trub cone settles)

| Valve | Position |
|-------|----------|
| Recirc | CLOSED |
| Whirlpool | OPEN → then CLOSED |
| Knockout | CLOSED |
| Pump | ON → then OFF |

## ❄️ Chill

1. Connect immersion chiller to cold water → start chilling
2. Target: **68°F**
3. Stir gently with chiller to speed cooling
4. Take **OG reading** when close to temp (target ~1.038-1.043)

## 🍺 Knockout to Fermenter

1. Fermenter sanitized (Star San spray, swirl, drain — no rinse)
2. Open Knockout valve — **1.25-1.5 turns only** (slow gravity drain)
3. Drain wort into fermenter — Pump stays OFF
4. Close Knockout when done
5. **Pitch M29** — sprinkle on top of wort
6. Seal fermenter → airlock/blow-off → set to **68°F**

| Valve | Position |
|-------|----------|
| Recirc | CLOSED |
| Whirlpool | CLOSED |
| Knockout | OPEN (1.25 turns) |
| Pump | OFF |

## 🧼 Clean Up (Do This Now!)

1. Rinse kettle with water
2. Remove pump filter + basket seal → rinse
3. Open all valves → run water through to flush lines
4. Heat 2 gal water to ~135°F → add PBW → recirculate all paths
5. Drain → rinse 2-3 times
6. Open all valves → flip upside down to dry

## 🍺 Kegging

- Cold crash 38°F for 24-48hr (optional)
- Sanitize keg → closed transfer
- Force carb: **20-22 PSI at 38°F** for 5-7 days
- Target: **3.0-3.2 volumes CO₂**
- Serving: reduce to ~12 PSI, use 8-10 ft beer lines
